*정규표현식(Regular expression의 약자 regiex 레직스) 정규식:텍스트에서 우리가 원하는 특정한 패턴을 찾을때 사용 (전화번호형태의 패턴을 찾거나 웹 사이트 주소형태의 패턴을 찾거나,이메일형태의 패턴을 찾을때 등등에 사용됨) 정규식은 문자를 검사하고 싶을때 사용된다. ⭕사용방법 :정규식은 /로 시작하여 "나는 정규표현식"임을 나타낸다 :/우리가 찾고자하는 패턴 = /안에 우리가 찾고자하는 패턴을 넣어주면 된다. :/regiex/i = i는 어떤 옵션에 다라 검색할건지 플래그를 활용함 ⭕문법 /color|you/ =color또는 you를 찾아라! | : 또는 (or) () : group ex : gr(e|a)y = gr로 시작하고 e또는a가 있고 y로 끝나는거 (?:):찾지만 기억하지는 않음 (그룹에 대한 정보를 기억하지 않음) ex : gr(a|e|d)y = gr로 시작고 y로끝나는데 a나e나d가 있는 거를 찾아라 [] : 괄호안의 어떤 문자던! ex: gr[aed]y = 대괄호안에 있는 글자중 하나라도 만족하면 출력해라 ex : gr[abcdefg]y =a,b,c,d,e,f,g,중 ex : gr[ag]y =a부터 g안에 있는 글자중 위예제랑 같은뜻임 ex: gr[a-zA-Z]y = a부터a까지 대문자던 소문자던 하나라도 만족하면 찾아라 ex:[a-zA-Z0-9] 모든문자,숫자 하나라도 만족하면 찾아라 ex : [^a-z] 소문자 a부터z가 아닌것들을 모두찾아라 ex :[a-zA-Z0-9] 모든문자,숫자가 아닌것들을 찾아라(특수문자 찾을때사용)
